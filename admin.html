Team Updates Admin

Add, edit or remove team announcements shown on the founder dashboard.
How to update:
1. Edit the updates below
2. Click Save & Copy JSON
3. Open fetch_data.py in your repo
4. Replace the TEAM_UPDATES list with the copied JSON
5. git add fetch_data.py && git commit -m "update team" && git push
Dashboard updates on next Action run (or trigger manually).
